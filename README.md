1. What is JSX, and why is it used?

ans: JSX (JavaScript XML) is a syntax extension for JavaScript used in React that allows you to write HTML-like code inside JavaScript.

2. What is the difference between State and Props?

ans. In React, State and Props are both used to store data, but they are used in different ways.

3. What is the useState hook, and how does it work?

ans: UseState is a Hook in React that allows functional components to store and update state.

4. How can you share state between components in React?

There are several ways to share state:

1. Lifting State Up.
2. Context API.
3. State Management Libraries.
